# 🚀 Age Calculator Web App

A modern and responsive Age Calculator built using **HTML, CSS, and JavaScript**.  
This project allows users to calculate their exact age in **years, months, and days** by selecting their birth date.

---

## ✨ Features

- 🎯 Instant age calculation
- 📅 Date picker input
- 🎨 Modern glassmorphism UI
- 🌈 Animated heading underline
- 📱 Fully responsive design
- ⚡ Smooth button interactions
- 💡 Beginner-friendly JavaScript project

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Animations)
- JavaScript (Date handling logic)
